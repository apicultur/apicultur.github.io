APIdays Mediterranea 2016

The web 1.0 was readable, the web 2.0 was social, now the web is programmable through application programming interfaces, aka apis.



APIdays is the two-day conference about the technology and business of APIs. It is the main independent conference on APIs held worldwide. APIdays Mediterranea is APIdays edition for the South of Europe, and it gathers together in Barcelona developers, IT managers, startups, corporations and those involved in the API industry to discuss, learn and share about the future and business of APIs.

Especial track on APIdays Mediterranea will be focusing on communication APIs : From Telco to chatbots.

Organizers

 

The API platform to consume and distribute APIs.

 

APICULTUR is the API platform to consume and distribute APIs. We are strongly committed to the API philosophy and work to make APIs accesible for everyone.

Webshell makes API integration easy for developers.


WEBSHELL develops tools for making API integration easier for developers on authentication, scripting and widgets, in Paris, and manages the {"apis":"the joy"} blog.



APIdays Mediterranea is an intitiative from Apicultur following the first edition of APIDays founded by Webshell and FABERNOVEL in 2012.
All the organizers, sponsors, speakers and attendees are asked to follow our code of conduct.


Mobile World Centre es una iniciativa público-privada creada por Mobile World Capital Barcelona (MWCB) y Telefónica con el objetivo de acercar a la ciudadanía el mundo de la telefonía móvil y de Internet. Es un espacio para descubrir cómo la transformación mobile está revolucionando las vías de comunicación, nuestro día a día o el entorno social y empresarial. El Centre es un espacio abierto a los ciudadanos que acoge una exposición permanente y ofrece una agenda de actividades vinculadas a la actualidad y a proyectos centrados en la transformación social, cultural, tecnológica y económica de la movilidad. Mobile World Centre es también un punto de información y difusión del resto de iniciativas de MWCB.
